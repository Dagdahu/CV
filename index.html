Projets

Robofriend

Description

Robofriend est le résultat d'un exercice guidé du premier cours Udemy (Zero to Mastery) visant dans un premier temps à découvrir le développement d'application react, puis d'y incorporer redux. Ce site permettait également de se familiariser aux api, puisqu'il y fait appel à deux reprises. La première fois pour récupérer une liste de noms, la deuxième pour associer à chaque nom une image de robot, celle-ci étant générée en se basant sur la chaine de caractères fournie en paramètre.

Le site ainsi que le code sont hebergés sur la plateforme GitHub.

SmartBrain

Description

SmartBrain est le projet final du premier cours Udemy (Zero to Mastery). Son but était de créer un site utilisant une bonne partie du savoir aquis au long du cours. En plus du site front-end, il introduit ainsi la notion de back-end : server et base de donnée (PostgreSQL). Un autre aspect intéréssant était d'aborder les notions d'identification en ligne et de gestion sécurisée des mots de passe.

Le code est disponible sur la plateforme GitHub. Le site et le serveur sont quant à eux herbergés sur la plateforme Heroku. Heroku est une solution gratuite qui a un désavantage, le site se met en sommeil au bout d'un certain temps : la première connexion au site peut donc prendre jusqu'à 30sec.

Shop List

Shop'
list

Description

Shop List est exactement ce que son nom laisse deviner : un site de liste de course. Il s'agit de mon premier projet personnel, dont le but principal était de vérifier mes aquis. Il est constitué d'un site frontal connecté à un serveur et une base de donnée (MySQL).

Les fonctionnalités voulues étaient de pouvoir créer plusieurs listes et de les supprimer, de pouvoir facilement ajouter ou retirer des articles, et enfin d'avoir une proposition des articles les plus souvent utilisés pour faciliter la création d'une nouvelle liste. Enfin, le site devait idéalement pouvoir fonctionner sur mobile.

Le code est disponible sur la plateforme GitHub. Le site et le serveur sont quant à eux herbergés sur la plateforme Heroku. Heroku est une solution gratuite qui a un désavantage, le site se met en sommeil au bout d'un certain temps : la première connexion au site peut donc prendre jusqu'à 30sec.
